Документация, драйвера, исходники находятся здесь - https://github.com/Archer1292/ChannelViewer.

Непосредственно для работы нужет файл ChannelViewer2, все остальное не обязательно загружать.

Для использования программы ChannelViewer:
1. Установить драйвер перед первым запуском программы:
	а. для Vista и следующих i-756x_1223_ driverinstaller. Файл в папке драйвера. 
		Подключить устройство по usb.
		Двойным кликом запустить драйвер и следовать инструкциям(нажать далее, далее..)
		После установки драйвера отключить и подключить устройство.
		Устройство готово к работе и запуска ChannelViewer2.	
		Если установить драйвер не получилось, может помогут файлы how to install.
	
	б. для win98, XP - http://ftp.icpdas.com/pub/cd/usb_tm/napdos/usbconverter/i-756x_series/win_xp_64-bit_driver/

2. Подключить устройство и запустить ChannelViewer2 из главной папки*.

3. Если программа не может подключиться к модулю, или снять результаты, можно проверить работоспособность модуля запустив утилиту DCON_Utility_Pro_PC_2107_0322(от производителя).
Находится в папке драйвера.

*это исполняемый архив, в котором находится программа и необходимые ей dll, при желании его можно разархивировать и запускать непосредственно приложение, 
но тогда нужно будет следить чтобы dll находились с ним в одной папке.
